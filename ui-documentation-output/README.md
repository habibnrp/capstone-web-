# Flood Monitoring UI Documentation

Hasil dokumentasi UI dari project ini disimpan di folder terpisah dari source code.

## Isi folder
- `docs/` - deskripsi tiap halaman
- `screenshots/` - screenshot tampilan aplikasi

## Halaman yang didokumentasikan
- Dashboard
- Data
- Admin Panel
- Login
